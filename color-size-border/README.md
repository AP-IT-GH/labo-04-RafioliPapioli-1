# color-size-border

* plaats een titel, ondertitel, 2 paragrafen en een afbeelding (reeds voorzien) x
* vul de stylesheet in en link hem aan de pagina x
  * maak de titel 32px groot x
  * maak de ondertitel 28px groot x
  * zet beide titels in de kleur "rebeccapurple" en het vet x
  * onderlijn enkel de ondertitel x
  * zet de paragrafen op 16px x
  * geef de afbeelding een gestipte zwarte rand van 5px dik x
